Qu'est-ce qu'une carte Amazon ?

Les cartes-cadeaux Amazon.com sont le moyen idéal de donner à vos proches exactement ce qu'ils espèrent, même si vous ne savez pas ce que c'est. Les destinataires peuvent choisir parmi des millions d'articles dans tout le magasin. La valeur stockée des cartes-cadeaux Amazon.com n'expire jamais.


Comment recevoir ma carte Amazon ?

Tout d'abord, choisissez le montant de crédit prépayé que vous souhaitez sur votre carte-cadeau. Après paiement, le code de votre carte cadeau s'affiche directement sur votre écran. Simultanément, vous recevez également un e-mail contenant le code, la facture et les instructions d'utilisation.


Comment utiliser une carte-cadeau Amazon?

Vous pouvez utiliser votre carte Amazon directement sur le site web d'Amazon.

  • Connectez-vous à votre compte Amazon.
  • Cliquez sur «Appliquer une carte-cadeau à votre compte».
  • Entrez votre code de réclamation et cliquez sur «Appliquer à votre compte».

Après avoir activé votre carte Amazon, elle est automatiquement ajoutée à votre compte. Besoin d'aide pour activer vos codes Amazon ? Contactez notre support.
