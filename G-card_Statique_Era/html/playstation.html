Qu'est-ce qu'une carte PlayStation Network ?

La carte PSN (carte PlayStation Network) est une carte cadeau avec laquelle vous pouvez recharger votre porte-monnaie PSN. Avec vos crédits PSN, vous pouvez commander des produits du PlayStation Store quand vous voulez, comme de nouveaux jeux PlayStation ou des offres alléchantes. Pas assez de crédits PSN ? Rechargez simplement vos crédits avec une carte cadeau PlayStation.


Comment recevoir ma carte PlayStation ?

Après avoir terminé votre commande, nous vous enverrons immédiatement votre carte cadeau PlayStation Store sous la forme d'un code numérique. Vous verrez ce code immédiatement sur votre écran et vous le recevrez également par e-mail. Vous recevez votre code avec des instructions, afin que vous puissiez toujours activer facilement votre code PSN dans le PS Store.


Comment activer mon code PSN ?

Vous pouvez utiliser votre carte PlayStation Network sur votre console PlayStation.

  • Sur l'écran d'accueil de votre PlayStation, allez sur "PlayStation Store".
  • Sélectionnez "Activer un code" dans le menu à gauche de votre écran.
  • Entrez votre code PSN.

Après avoir activé votre carte prépayée PSN ou carte cadeau PS Plus, elle est automatiquement ajoutée à votre compte. Besoin d'aide pour activer vos codes PlayStation ? Contactez notre support.
