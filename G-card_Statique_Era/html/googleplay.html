Avec une carte cadeau Google Play, vous pouvez jouir de tout le contenu disponible dans le Play Store. La carte est également utilisable pour le service de musique Google Play Music.

Acheter une carte Google Play sur G-Card

Les cartes cadeaux Google Play vous permettent de rapidement recharger votre compte Google Play. Sur G-Card, vous pouvez facilement acheter des cartes Google Play en ligne sans frais supplémentaires. Vous n'avez pas besoin d'une carte de crédit et la carte prépayée Google vous est immédiatement livrée, vous pouvez donc utiliser immédiatement le code numérique.


Qu'est-ce qu'une carte cadeau Google Play ?

Une carte cadeau Google Play est une carte prépayée avec laquelle vous pouvez effectuer des achats sur Play Store. Vous pouvez choisir des cartes prépayées Google Play de différents montants : 15 euros, 25 euros et 50 euros. De plus, vous pouvez également acheter des cartes Google Play au montant variable. Ces cartes cadeaux Google Play vous permettent de choisir une valeur entre 15 et 150 euros. Vous pouvez utiliser le code Google Play Store vous-même ou l'offrir à une tierce personne. Le carte Play Store n'a pas de date d'expiration.


Comment obtenir ma carte Google Play ?

Vous obtenez votre carte Play Store directement après avoir effectué votre commande sur notre site sous la forme d'un code numérique. Ce code sera affiché sur votre écran et vous le recevrez également sur votre boîte mail. Le code cadeau Google Play que vous recevrez comprendra les instructions d'utilisation, activer votre code Play Store est donc toujours simple.


Comment utiliser mon code Google Play ?

Vous pouvez utiliser votre carte Google Play de différentes façons :

  • Cliquez sur le code et suivez la procédure indiquée pour automatiquement recharger votre crédit Google Play.
  • Vous pouvez également activer votre code sur play.google.com/redeem.
  • Enfin, vous pouvez activer votre code dans l'application Play Store. Dans le menu de l'application Play Store, sélectionnez 'utiliser un code'.

La valeur de votre code est directement ajoutée à votre crédit Google Play Store. Vous avez besoin d'aide pour activer votre carte Google Play ? Notre support est à votre disposition sept jours sur sept.
