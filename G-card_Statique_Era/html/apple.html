Les cartes cadeaux iTunes sont le cadeau idéal pour les utilisateurs d'Apple, leur donnant accès à tout le contenu de la index iTunes. Envoyez instantanément des cartes cadeaux Apple avec G-Card.

Acheter une carte cadeau Apple Store et iTunes sur G-Card

Vous souhaitez acheter des cartes cadeaux App Store et iTunes ? Choisissez alors d'acheter votre carte iTunes en ligne sur G-Card. Lorsque vous achetez votre carte Apple sur G-Card, le code numérique vous est instantanément envoyé.


Qu'est-ce qu'une carte cadeau Apple Store et iTunes ?

Une carte Apple Store et iTunes est une carte prépayée contenant des crédits qui peuvent être utilisés pour effectuer vos achats dans l'App Store d'Apple. G-Card propose une large variété de cartes cadeaux iTunes, d'une valeur de 15 euros, 25 euros et 50 euros. Vous pouvez utiliser la carte iTunes vous-même, ou choisir de l'offrir à un ami ou à un membre de votre famille.


Comment recevoir ma carte prepayée iTunes ?

Après avoir acheté une carte iTunes, votre code iTunes apparaîtra directement sur votre écran et vous le recevrez également dans votre boîte mail. Ne partagez pas ce code avec des inconnus.


Comment activer ma carte iTunes ?

Après avoir reçu le code de votre carte cadeau iTunes, vous devrez l'activer dans l'App Store.

  • Pour activer la carte prepayée iTunes, cliquez simplement sur le code.
  • Vous pouvez également activer le code iTunes sur appstore.com/redeem.
  • Après avoir activé le code, le montant de la carte cadeau sera automatiquement ajouté à votre solde iTunes.

Vous pouvez immédiatement utiliser vos nouveaux crédits App Store et iTunes.
